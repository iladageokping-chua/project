ส่วนที่ 1 การใช้งาน
   1.1 เปิด XAMPP จากนั้นกด start Apache และ MySQL 
   1.2 แตกไฟล์ และ ลาก Folder เข้าไปในใน C:\xampp\htdocs
   1.3 เปิด folder ใน VS Code หรือโปรแกรมอื่นๆ
   1.3 พิมพ์ npm install ใน Terminal
   1.4 พิมพ์ npm start เพื่อเริ่มโครงงาน ( ควร cd เข้าไปใน folder ทุกครั้ง )
   
ส่วนที่ 2 การติดตั้งโครงงาน
   2.1 โปรแกรมที่จำเป็นต้องมี คือ XAMPP
   2.2 สร้าง Database ชื่อ project_cp151 และ
   import project_cp151.sql สำหรับสร้างฐานข้อมูลใน phpmyadmin
   2.3 ติดตั้ง PHP และ Node.js
